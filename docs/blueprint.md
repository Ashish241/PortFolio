# **App Name**: Astrofolio

## Core Features:

- Personal Information Display: Display name, tagline, bio, and photo.
- Project Showcase: Showcase projects with descriptions, tech stack, and results.
- Skills Highlighting: Display technical and soft skills.
- Education Timeline: Show education history in a timeline format.
- Contact Form: Enable users to send messages directly through the website.
- Resume Tool: Leverage user profile information and project details to allow a generative AI tool to provide suggestions for how to optimize their resume for a target job, based on details pasted into a text area. Generates updated resume for download.
- Google Analytics Integration: Track website traffic and user behavior using Google Analytics.

## Style Guidelines:

- Primary color: Deep navy (#0F172A) for a modern and professional look.
- Background color: Soft white (#F8FAFC) for a clean backdrop.
- Accent color: Darker Indigo (#3C37A2) for CTAs and highlights to add visual interest while keeping the palette professional.
- Headings font: 'Poppins' (sans-serif) for a modern and elegant feel. Note: currently only Google Fonts are supported.
- Body font: 'Inter' (sans-serif) for clean and readable content. Note: currently only Google Fonts are supported.
- Simple line icons for a clean and modern aesthetic.
- 3-column responsive grid for project cards.
- Section spacing of 80-120px for clear content separation.
- Smooth micro-animations and page transitions for an engaging user experience.
